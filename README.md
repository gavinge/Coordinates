
# 🌟 项目概述

这是一个探索AI时代编程新范式的实验性项目。在大型语言模型快速发展的今天，我们尝试采用"vibe-coding"这一新兴的开发方式，将创意快速转化为可工作的软件。本项目充分证明了：在AI的辅助下，一个有想法的开发者完全可以组建一个"一个人的团队"，完成从概念到实现的完整开发流程。

# 🎯 核心特性

## 1. Vibe-Coding 开发实践

本项目主要采用传统的vibe-coding方式进行辅助编程。所谓vibe-coding，是指开发者与AI编程助手紧密协作，通过自然语言描述需求、讨论设计方案、生成代码框架、优化实现细节的开发模式。这种开发方式让我们能够：

- 快速验证产品想法
- 降低技术实现的认知负担
- 保持开发节奏和创作灵感
- 将更多精力投入到核心业务逻辑

## 2. 复杂的Qt UI实现

项目基于Qt框架构建了功能丰富的用户界面，充分利用Qt的跨平台特性和丰富的组件库。UI设计注重用户体验，实现了流畅的交互效果和专业的视觉呈现。

## 3. 自研MVC架构

为了保持代码的整洁性和可维护性，我们自主研发了轻量级的MVC（Model-View-Controller）框架。这一框架：

- 清晰分离业务逻辑、数据显示和用户交互
- 降低模块间耦合度，提高代码复用性
- 使项目结构更加清晰，便于后续功能扩展
- 为复杂的UI交互提供良好的架构支撑

# 🔧 编译环境

## 必要组件

- **Qt版本**：5.15.2
- **编译器**：Visual Studio 2017

## 编译步骤

1. 安装Qt 5.15.2及对应的VS开发工具
2. 使用Qt Creator或VS2017打开项目文件
3. 配置正确的Qt版本路径
4. 执行qmake生成Makefile
5. 编译构建项目
